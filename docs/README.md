# Documentation

This folder documents the repository as it exists today, with separate notes on
the work that is still planned.

## Start Here

- [Architecture](architecture.md): How the current compiler pieces fit together
- [Calibration Guide](calibration.md): Representative dataset sampling,
  statistics, and current calibration behaviour
- [Development Guide](development.md): Day-to-day workflow, verification, and
  extension points
- [Environment Setup](environment-setup.md): Local environment and hook setup
- [Roadmap](roadmap.md): Near-term and medium-term future efforts

## Reading Order

If you are new to the repo, a good path is:

1. Read [Architecture](architecture.md) for the big picture.
2. Read [Development Guide](development.md) to understand how changes are made.
3. Read [Calibration Guide](calibration.md) if you are working on
   quantization-related flows.
4. Use [Roadmap](roadmap.md) to separate current capability from planned work.
